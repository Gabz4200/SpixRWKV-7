Compare Vision RWKV-7 vs ViT speed (alt ViT)

options:
  -h, --help            show this help message and exit
  --device {cpu,cuda,auto}
                        Device to run on (default: auto-detect)
  --warmup WARMUP       Number of warmup runs (default: 5)
  --runs RUNS           Number of timed runs (default: 20)
  --batch-size BATCH_SIZE
                        Batch size for inference (default: 1)
  --skip-large          Skip large config (slow on CPU)
  --resolution-sweep RESOLUTION_SWEEP [RESOLUTION_SWEEP ...]
                        Image sizes for resolution sweep (default: 64 128 512)
  --sweep-size {tiny,small,medium,large}
                        Model size for resolution sweep (default: small)
  --model-type {spix,vq,conv}
                        Backbone type
  --img-size IMG_SIZE   Input image height in pixels (proportional width;
                        default: 512)
  --compare-variants COMPARE_VARIANTS [COMPARE_VARIANTS ...]
                        If set, compare these variants head-to-head per size,
                        e.g. spix conv vq
  --downsample-factors DOWNSAMPLE_FACTORS [DOWNSAMPLE_FACTORS ...]
                        Downsample factors for spix/base model builder
                        (default: 1.0)
